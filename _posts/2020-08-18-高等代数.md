# 高等代数

## 基本Notation

* Cardinality of X: $$ \vert X \vert $$ 这个set里有多少个元素

* $$\mathbb{Z} = \{ \cdots, -2, -1, 0, 1, 2, \cdots \}$$ 

* $$\mathbb{Z}_{\ge 0} = \{ \cdots, -2, -1, 0, 1, 2, \cdots \}$$ 

* $$\mathbb{N} = \{ 1, 2, \cdots \}$$ 

* $$ \mathbb{Q} = \{ {\frac{m}{n} | m,n \in \mathbb{Z} }  \} $$
